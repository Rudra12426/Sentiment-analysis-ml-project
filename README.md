# Sentiment Analysis using Machine Learning

## Project Overview

This project implements a **Sentiment Analysis model** that classifies movie reviews as **Positive or Negative** using Natural Language Processing and Machine Learning techniques.

The system processes raw text reviews and predicts sentiment using **TF-IDF vectorization** and **Logistic Regression**.

---

## Features

- Text preprocessing
- Tokenization using `split()`
- Punctuation removal
- TF-IDF feature extraction
- Logistic Regression classifier
- Sentiment prediction

---

## Machine Learning Pipeline

Text Input  
↓  
Text Cleaning  
↓  
Tokenization  
↓  
TF-IDF Vectorization  
↓  
Model Training  
↓  
Sentiment Prediction
